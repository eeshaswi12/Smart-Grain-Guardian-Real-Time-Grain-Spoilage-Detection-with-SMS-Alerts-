# Smart-Grain-Guardian-Real-Time-Grain-Spoilage-Detection-with-SMS-Alerts-
Grain storage is a critical aspect of the agricultural industry, requiring constant monitoring to 
ensure optimal conditions and prevent spoilage. This paper presents a real-time grain 
monitoring system designed to provide continuous oversight of storage environments. The 
system leverages a combination of sensors and IoT technology to collect data on temperature, 
humidity, and gas levels, all key indicators of grain quality and storage stability. The use of the 
Arduino Uno board, in conjunction with the MQ135 gas sensor, DHT22 temperature and 
humidity sensor, and TCS2300 color sensor, allows for efficient data acquisition and 
processing. Through a GSM module, the system sends automated alerts to stakeholders via 
SMS when predefined thresholds are breached, indicating potential risks of spoilage or 
contamination. The system operates on a periodic loop, monitoring conditions at regular 
intervals and triggering alerts when necessary. Threshold values for temperature, humidity, 
and gas levels are calibrated to ensure timely notification, allowing for proactive measures to 
prevent grain spoilage. This real-time grain monitoring system offers an innovative approach 
to agricultural storage management, addressing the need for reliable and scalable solutions in 
the agricultural sector. By providing a comprehensive monitoring system, this project 
contributes to reducing post-harvest losses and enhancing food security. The insights gained 
from this system can inform best practices for grain storage, promoting efficiency and 
sustainability in agriculture.
